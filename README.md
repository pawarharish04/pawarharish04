### ❖ Selected Works

#### 01. E-FIR Portal
**The Challenge:** Digitalizing law enforcement with zero compromise on integrity.
<br>
**The Engineering:**
* **Immutable Security:** Implemented `JWT` stateless authentication with granular **RBAC**, ensuring data access is strictly role-dependent.
* **Optimized Schema:** Designed a normalized **PostgreSQL** database to handle relational complexity with sub-millisecond query performance.
* **Visual Clarity:** Developed a **TailwindCSS** dashboard that converts raw crime data into actionable, real-time metrics.

> `React` • `Node.js` • `PostgreSQL` • `Security First Architecture`

<br />

#### 02. Real-Time Emergency Triage
**The Challenge:** Reducing critical decision time in emergency wards.
<br>
**The Engineering:**
* **Algorithmic Efficiency:** Engineered a custom **Heap-based Priority Queue** (DSA) to automate patient sorting, removing human error from triage.
* **Concurrency Control:** Solved race conditions between doctors and receptionists using optimistic locking and real-time state synchronization.
* **Impact:** Reduces patient wait-time variance through mathematical sorting.

> `React` • `Express` • `Advanced DSA` • `Real-Time Systems`

<br />

#### 03. Tribe
**The Challenge:** Social discovery without the invasion of privacy.
<br>
**The Engineering:**
* **Privacy Engineering:** Built a geospatial filtering engine that calculates distance *without* exposing precise user coordinates.
* **Type Safety:** Leveraged **Next.js** and **Prisma** for end-to-end type safety, eliminating a class of runtime errors before they ship.
* **Trust Layer:** Architecture designed around "Privacy Toggles," giving users absolute control over their data visibility.

> `Next.js` • `Prisma` • `PostgreSQL` • `Privacy by Design`
